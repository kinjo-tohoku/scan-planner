Help — filman scan planner

Plan a triple-axis scan in (Q, E), check whether every point is reachable within the motor soft-limits, and export a filman .scn script. No resolution calculation; configuration is entered by hand.

Workflow

  • 1. Sample — real-space lattice a,b,c,α,β,γ and the two orienting reflections u, v that define the scattering plane.
  • 2. Spectrometer — fixed ki/kf + value, and the analyzer (PG002/PG004). GPTAS defaults are filled in.
  • 3. Soft limits — sample motors C2 (ω) and A2 (2θ). With kf fixed these are what actually bound a scan.
  • 4. Scan definition — single line or map (below).
  • 5. Evaluate → green = reachable, red × = not, with the reachable region shaded behind (on constant-E h,k views). Generate .scn writes the filman script.

Start / End / Δ — NP is computed

You enter start, end, and step Δ per axis. The number of points is derived: NP = floor((end − start) / Δ) + 1 on the swept axis (any axis with Δ≠0). Axes with Δ=0 stay fixed at their start value, so their end field is ignored. The computed NP is shown live under the grid.

Map types

Maps sweep along the scattering plane directions u, v (the two orienting reflections), so any plane works — e.g. u=(1 1 0), v=(0 0 1) is the HHL plane. You give the start point plus an extent + Δ for the inner and outer sweeps. Default u=(1 0 0), v=(0 1 0) is the usual h×k map.

  • Plane u×v (const E) — a grid in the plane: inner sweep runs along u, repeated for each step along v. The start point must lie in the plane.
  • Dispersion (along u/v) ×E — inner sweep is E at constant Q; the outer shift moves along u or v (pick with “disperse along”). The plot x-axis is the reduced coordinate ξ along that direction.

Each line becomes one NS= scan; the map runs as a combined GO 1-N.

Auto-fit to reachable region

When ON, points outside the reachable region are dropped and each line is trimmed to its reachable run (split if a gap is in the middle). The summary then reads e.g. reachable 62/62 · 43 dropped — the dropped count tells you points were removed, and they appear as faint red × on the plot. When OFF you get the full rectangle.

Long maps — GO blocks of 40

filman holds at most 40 scan slots per GO. If a map has more lines, the .scn is split into blocks — each block re-numbers NS=1..40 and ends with its own GO 1-40 — so the whole map still runs from one file.

Generated .scn

Lattice / spectrometer / RS are assumed set & tuned beforehand and are not written. Only scan definitions (NS/HS/KS/ES/DE/DH/DK/NP/MN[/NT], comma-separated) and GO are emitted. filman is in-plane (h,k), so the scan l is ignored (out-of-plane = unreachable).
